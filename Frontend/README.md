## MeetMe - Связываем миры, создавая встречи
Проект разработан на TulaHack 2023.

## Как запустить

Для запуска нужно:

```bash
npm i
# next
npm run build
# next
npm run start
```

## Стэк
- Next.js 13 (App Router)
- Redux Toolkit
- Tailwind
- React Query
- React Hook Form
